**Sketch:**
Sketches allow you to draw a 2d shape because all 3d models start from 2d sketches

**Features:**
Features are what allow you to turn the 2d shape you drew into a 3d model using various tools such as extrude and revolved base

**2D Sketch**
Every 2D sketch needs to be made on a plane. If you were asked to draw something you would need a piece of paper to draw on (the plane
- Solid works gives you 3 planes you can work with which are the front plane, the top plane and the right plane

**Smart dimension:**
Smart dimension allows you to change the dimensions in a controlled manner
- When you don't select smart dimension you're able to freely change how a shape looks
- A smart dimension defined shape will not change 
- Try and stay in the **full defined state** as much as you can to avoid too many unwanted changes
- **Underdefined:** blue state
- **Over defined:** red state

**Blue box selection (left to right)**
Used for more specific selections

**Green box selection (right to left)**
Used for selecting full shapes without having to cover the entire shape

**Rotating Component:**
- Scroll wheel
- Arrow keys
- Shift plus move

**Rebuilding:**
Editing your 3D build after building the sketch

**Drawing constraints:**
You can either draw on a flat surface of a component or on a 2d plane. Never on a corner or on and edge

**Tips:**
Always draw your components in the middle of the entire 3D space

**Ways to make sketches fully defined:**
- You can make a sketch fully defined by either using smart dimension by assigning dimensions or assigning sketch relations (vertical, horizontal, coincidental)
	- Entities include points, lines, surfaces
- Some sketch relations are applied manually without defining anything
