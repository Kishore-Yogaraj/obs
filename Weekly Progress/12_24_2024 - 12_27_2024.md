- **Strings in Java script, their different syntaxes and how to use them**
	- Single quotes
		- Most commonly used 
		- Can be used to print strings
	- Double quotes
		- Not as commonly used but can still be used when needed 
		- Double quotes and single quotes are interchangeable
	- Back ticks and Interpolation
		- Back ticks are used when we want to interpolate (insert a value into a string)
		- Back ticks are also known as template strings
		- It is done as shown below
```javascript
	const variable1 = 10
	console.log('This is the number: ${variable1}')
```

- Concatenating
	- You can add strings and numbers together to combine values
	- Example:
```javascript
'some' + 'more' + 'text'
```

	- Just remember that \n starts a new line of text
