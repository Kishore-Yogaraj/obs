### Solution
```python
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        ptr = head
        prev = None

        while ptr:
            tmp = ptr.next
            ptr.next = prev
            prev = ptr
            ptr = tmp
        return prev
```
