This is honestly a pretty logical question that can be solved using **two pointers**

Couple of things to keep in mind here. 

First we make sure we start our two pointers on **opposite ends** of the array which represent the heights of the tank

Then we enter our while loop and we calculate the area by doing the following:
`area = min(heights[l], heights[r]) * (r-l)`

after calculating the area, we need to find a way to keep track of the largest area we've seen so far. To do this we can us a **max function counter**. That essentially looks like this:
`res = max(area, res)`

This means that every time we calculate the area, we will always hold onto the max area using the res variable.

After that we need to optimize the size of the container which we can do by checking which height is greater of the left or right pointer.

If the left pointer is greater than we move the right pointer and recalculate max area, and if the right pointer is greater than we move the left pointer and recalculate the area

### Solution

```python
class Solution:
    def maxArea(self, height: List[int]) -> int:
        L = 0
        R = len(height) - 1
        res = 0

        while L < R:
            area = min(height[L], height[R]) * (R-L)
            res = max(res, area)
            if height[L] < height[R]:
                L += 1
            else:
                R -= 1
        return res
```
