### Here was my attempted solution

```python
class Solution:
    def countGoodSubstrings(self, s: str) -> int:
        # This is a fixed sliding window question
        # As a base case
        window = {}
        good_arr = []
        L = 0
        
        for R in range(len(s)):
            if R - L + 1 > 3:
                window.remove(s[L])
                L += 1
            
            # Add values to set for each iteration

            # If the set is full, then it is a good string
```

The problem here is that with a test case where the same letter follows another letter, for example a test case like this: `aababcabc` we run into an issue where we are attempting to remove the second a because the window is larger than 3 but the second a was never added because you can't have duplicates in a set. If there is nothing to remove, you'll get an error. Your solution fails if you have 2 of the same letters beside each other

Let me walk through exactly what happens with a concrete example using the string "aabc":

1. Start with empty set: `window = {}`
2. R=0: Add 'a' to set → `window = {'a'}`
3. R=1: Add another 'a' to set, but since sets don't allow duplicates, the set doesn't change → `window = {'a'}`
4. R=2: Add 'b' to set → `window = {'a', 'b'}`
5. R=3: Add 'c' to set → `window = {'a', 'b', 'c'}`
    - Now R-L+1 = 3-0+1 = 4, which is > 3
    - Try to remove s[L] which is 'a' from the window set
    - Remove 'a' from the set → `window = {'b', 'c'}`
    - Increment L to 1
6. Next iteration (R=4 if the string were longer):
    - R-L+1 = 4-1+1 = 4, which is > 3
    - Try to remove s[L] which is 'a' again from the window set
    - But 'a' was already removed in the previous step!
    - KeyError: 'a' because 'a' is no longer in the set

The fundamental issue is that the sliding window conceptually contains "aabc" at that point, but the set only contains {'a', 'b', 'c'} because sets automatically deduplicate. When the window slides forward, the code tries to remove the second 'a' from the set, but there's only one 'a' in the set (which was already removed).

### How to solve this

I think you just overcomplicated this problem and stuck too much to your formulas. Unfortunately there is no proper way to solve any problem. You just have to recognize what the pattern is and attempt to implement it.

In this case we basically set our pointers up in the size of the window.

We loop until the right pointer reaches the end

While the window is still valid we check to see if any of the elements in the subarray are duplicates of one another.

If they aren't, then add to the count.

If they are increment the window

Return the count at the end,

```python
class Solution:
    def countGoodSubstrings(self, s: str) -> int:
        l=0 
        r=2
        count=0
        while r<len(s):
            if s[l]!=s[l+1] and s[l]!=s[r] and s[l+1]!=s[r]:
                count+=1
            l+=1
            r+=1
        return count
```
