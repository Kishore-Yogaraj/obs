What is a function?
- A function lets us reuse code

Rules for function names
- Can't use function
- Can't start with number
- Can't use special characters except & and _
- Best practice is to use camelCase

Functions create a scope
- Scopes limit where a variable exists
- Any variable created between the curly brackets only exists inside the curly brackets

Return Statement
- Lets us get a value out of a function
- We can return anything that results in a value

Tracing the code
- Following code in the code
- If you see a function you go see what that function is doing and then you follow it back to the return statements

Parameters
- Parameters put a value into a function

Functions can be called inside of other functions
