What is an object?
- An object groups multiple values together

Why do we use objects?
- Makes code more organized
- Used for grouping related values together
-  Let us group multiple values together and let us use multiple values together

JSON
- Javascript Object Notation
- a syntax
- similar syntax to Javascript object syntax with less features
- All values must have double quotes
- Can't use functions in JSON

Why use JSON
- JSON syntax can be understood by any programming language
- We use JSON when we send data between computers
- We use JSON when we store data

Built-in JSON object
- Converts Javascript object to JSON

Local Storage
- Variables are temporary
- If we refresh/close the page, all the variables are deleted
